******2.CustomComboBox 继承ComboBox ******
ComboBoxListBackground 下拉框背景颜色

******3.CustomPasswordBox 继承Control******

IsShowPassword 密码是否可见（默认false）
Password 明文密码(依赖属性)
CornerRadius
PasswordChar
PasswordPlaceHolder占位符提示

******4.CustomTextControl 继承TextBox（文本控件）******
CornerRadius
TextPlaceHolder占位符提示
TextInputType 输入文本类型 
{
    defaultText = 0,//普通文本(包括汉字)
    digit = 1, //仅实数
    letter = 2,  //仅字母
    digitOrLetterLine = 3, //数字、字母或下划线
    digitAndLetter = 4, //数字和字母(密码使用)
    chinese = 5,  //汉字
}
